# Laravel Zoho Books

 This Laravel package simplifies integration with Zoho Books Accounting System, streamlining API interactions for easier accounting management.

## Requirements

| Software          | Version           |
| :---------------- | :---------------- |
| `php`             | `^8.2`            |
| `Composer`        | `^2.3`            |
| `Laravel`         | `^11.0`           |
